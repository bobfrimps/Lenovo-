WHAT THIS PACKAGE DOES

  This package updates the firmware (Lenovo Thunderbolt 3 Essential dock) to update the
  following chip of dock.

  Chip                	Chip name in the Dock
  ------------------------------------------------------------------------------
  Intel TBT3		TITAN-RIDGE_DD
  Synaptics DP HUB	VMM5210
  VIA USB HUB1		VL820
  VIA USB HUB2		VL817
  Conexant Audio	CX21985
  Cypress DMC		CY7C65219
  Cypress PD		CCG5C

  Refer to marketing materials to find out what computer models support which
  dock.

  If this package has been installed, this package will fix problems, add new 
  functions, or expand functions as noted below.

  This program is language independent and can be used with any language system.

INSTALLATION INSTRUCTIONS

Notes:
- If your dock runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Click once on the underlined file name(tbtesdkfw10XX_X.exe). Once this is done, some pop-up windows will appear.
  2. Follow the instructions on the screen.
  3. Choose the folder you would like to download the file to and click Save.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating that the download completed successfully.

  4. Make sure to be logged on with an administrator account.
  5. Locate the folder where the file was downloaded.

  method 1:
  6. For UI mode [File downloaded folder\]tbtesdkfw10XX_X.exe /tvsuui
  7. Follow the instructions on the screen to complete update and restart the computer.
  8. For Silent mode [File downloaded folder\]tbtesdkfw10XX_X.exe /u /VERYSILENT

 
  method 2:
  Extracting file
  9. For Extracting file [File downloaded folder\]tbtesdkfw10XX_X.exe /unp /DIR=c:\temp\EsDockFwUpdate
 10. If you would like to change a different folder, please modify /DIR=[Select folder].
 11. Follow the instructions on the screen.
 12. In the "Extract Or Install" window, click "Extract Only". All the necessary files will
     be extracted to the folder selected in the step 9. (Ex: c:\temp\EsDockFwUpdate)

  Update FW file
 13. UI mode: At the command line mode and execute [File extracted folder\]Fwupdate.exe
 14. Silent mode: At the command line mode and execute [File extracted folder\]Fwupdate.exe /u /s /tvsu
 15. Silent mode force reboot system: At the command line mode and execute [File extracted folder\]Fwupdate.exe /s /forcereboot

  Check dock/package version
 16. At the command line mode and execute [File extracted folder\]Fwupdate.exe /showver
