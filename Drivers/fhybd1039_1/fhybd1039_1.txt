WHAT THIS PACKAGE DOES

  This package updates the firmware (ThinkPad Hybrid USB-C with USB-A dock) to update the
  following chip of dock.

  Chip                     	 
  ------------------------------------------------------------------------------
  USB HUB
  PD/Billboard

  Refer to marketing materials to find out what computer models support which
  dock.

  If this package has been installed, this package will fix problems, and add new 
  functions, or expand functions as noted below.

  This program is language-independent and can be used with any language system.

INSTALLATION INSTRUCTIONS

Notes:
- If your dock runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.


  Method 1: Package Install
  (Install option) 
-- Installation by the one-click update:
	1. Connect the dock to PC/NB.
	2. Double-click fhybd1039_1.exe .
	3. Follow the instructions on the screen, and choose "Install".
	4. Press the "Update" button of the FW update tool.

  Method 2: Silent Install
  (Extract only option)

-- Installation by command-line manually update:
	1. Connect the dock to PC/NB.
	2. Double-click fhybd1039_1.exe .
	3. Follow the instructions on the screen, and choose "Extract only".
	4. In the "Select Destination Location" window, click Extract. If you would like to select a different folder, click Browse.
	5. In the Ready to Install window, click Install. All the necessary files will be
	extracted to the folder you selected. (Default: C:\DRIVERS\HybridDockFwUpdateTool\[YYYYDDMM].[TIMESTAMP])
	
-- Silent command list
    /? : Display this help message
    /v : Show FW version on screen
    /u : Update firmware
    /s : Silent mode
    /f : Force update firmware


-- Open a Command Prompt with Administrator privileges for the following --

-- Silent FW version check by command:

	For checking current FW version usage: [File extracted folder\]Fwupdate.exe /v

    Return Code: 
        1: Checking version successfully.
       -1: Checking version failed due to dock not found or other unknown reason.
     -101: Checking version failed due to failure on USB hub chip.
     -102: Checking version failed due to failure on PD/Billboard chip.	   
	 
-- Silent FW update by command:

	For updating FW usage: [File extracted folder\]Fwupdate.exe /u
	(Follow prompt instructions on the screen to complete the update.)
	
	For updating FW without prompt usage: [File extracted folder\]Fwupdate.exe /u /s

	For force updating FW in silent mode usage: [File extracted folder\]Fwupdate.exe /u /s /f 
     
    Return Code:
         0: Updating version successfully.
         1: Did not start updating process due to firmware version is newer or equal.
        -1: Updating version failed due to dock not found or other unknown reason.
    0x8001: The update failed due to failure on USB hub chip.
    0x8002: The update failed due to failure on PD/Billboard chip.
    0x9003: Battery < 25%. System battery power is less than 25%.
    036865: This process requires power to avoid an accidental power-off during an update.

SCCM Check version and Update
==============================
-- Silent mode check version by command:
     1. Connect the dock to PC/NB.
     2. [File download folder\]fhybd1039_1.exe /VERYSILENT /v 

-- Silent mode force update by command:
     1. Connect the dock to PC/NB.
     2. [File download folder\]fhybd1039_1.exe /VERYSILENT /u /f


Check version return codes for SCCM mode
========================================
-- Check Firmware:
     0: Checking version successfully, firmware update is not needed (dock's firmware is equal to or newer than package's).
     1: Checking version successfully, firmware update is needed (dock's firmware is older than package's).
    -1: Checking version failed due to dock not found or other unknown reason.

Update return codes for SCCM mode
========================================
-- Update Firmware:
     0: Updating version successfully.
     1: Did not start updating process due to firmware version is newer or equal.
   259: Updating version failed due to dock not found or other unknown reason.
